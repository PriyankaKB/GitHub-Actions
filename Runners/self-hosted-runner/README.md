# Self Hosted Runners

Setting up a self hosted runner:

1. Go to Repository --> Settings --> Actions --> Runners and click "New self-hosted runner":
<img width="1360" height="690" alt="image" src="https://github.com/user-attachments/assets/5ec4b3df-7f0f-49ce-95f7-2750ad276a4b" />


2. Select runner image:
<img width="1358" height="648" alt="image" src="https://github.com/user-attachments/assets/0ce266bb-c9eb-4426-96fb-402552602c02" />


3. Follow the instructions for setting up the runner for your selected image in your virtual machine.

**Note:** It is better to use a Virtual Machine on cloud for setting up a self-hosted runner.



**Reference:**
https://docs.github.com/en/actions/how-tos/manage-runners/self-hosted-runners/add-runners
