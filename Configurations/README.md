# Configure a GitHub Actions workflow

Events that trigger a workflow:

1. Scheduled events - Cron jobs - schedule
2. Manual events - workflow_dispatch
3. Webhook events - check_run
4. Repository dispatch events - repository_dispatch
5. Repository events - push, pull_request, release etc.

There are many more other events that trigger a workflow. Below reference can help you get into the details of each event type.  


**Reference:**
https://docs.github.com/en/actions/reference/workflows-and-actions/events-that-trigger-workflows
